# HEA Coating Machine Learning Framework

This repository contains datasets and machine learning codes used in:

"Decoding Phase Constitution, Hardness and Wear Behaviour in High-Entropy Alloy Coatings on Titanium Alloys through Intelligent Ensemble Learning"

Contents:
- Phase prediction dataset
- Hardness prediction dataset
- Wear prediction dataset
- Phase prediction code
- Hardness prediction code
- Wear rate prediction code
